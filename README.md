# DeadFocus

Dead Focus is a .NET MVC web appliction writen in C# on Visual Studio and currently stores data using SQLite.
This and open-source project that aims to help users stay focused and productive by minimizing distractions and staying on one track. 
DeadFocus provides a set of tools and features designed record long and short term goals, track productivity and progess, and commit certain amounts of time to each goal base on its prioity in your life.

## Planned Features

 - Dashboard: A central hub where users can view their goals, progress, and productivity metrics at a glance.
 - Add Goals: Users can create and manage both long-term and short-term goals, commiting an amount of time per week, and setting priorities and deadlines for each.
 - Edit Goals: Users can edit their existing goals, update progress, and adjust priorities as needed.
 - Begin Focus Sessions: Users can start focus sessions for specific goals, during which distractions are minimized, and time is tracked. Users can set timers for these sessions and receive notifications when the session ends. As well as a comment about why or why not the session was successful.


## Technologies Used

 - .NET MVC Framework
 - C# Programming Language
 - SQLite Database
 - Visual Studio IDE
 - HTML, CSS, JavaScript for front-end development
